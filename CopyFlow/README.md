# Copyflow｜抄写助手

手抄长篇文本时的辅助定位工具，防止抄错行、多抄。三行递进式显示，让你始终清楚当前抄到哪里。

## 功能

- **三行递进显示** — 同时展示上一句、当前句、下一句，按回车逐句递进
- **Markdown 自动清理** — 自动移除 AI 生成文本中的 `**`、`##`、`---`、`[]()` 等 Markdown 语法字符
- **原文地图** — 全文模糊铺底，当前句高亮居中，全局位置一目了然
- **深浅色模式** — 支持亮色/暗色切换，默认跟随系统偏好
- **智能断句** — 句末标点优先断句，超过最大字数强制切分
- **简约模式** — 隐藏背景特效，降低低性能设备渲染开销
- **移动端适配** — 响应式布局，小屏设备显示触控按钮
- **点击递进** — 鼠标点击页面空白处即可前进
- **页码跳转** — 点击页码输入数字，回车直达任意句子
- **设置持久化** — 所有偏好自动保存至本地浏览器

## 快捷键

| 按键 | 操作 |
|------|------|
| `Enter` / `空格` | 递进到下一句 |
| `Backspace` / `←` | 回退到上一句 |
| `Esc` | 重置，回到输入文章界面 |
| 点击页码 | 输入数字跳转到指定句子 |

## 使用方式

1. 打开页面，粘贴要抄写的文本
2. 点击「开始」
3. 看着大字跟着抄，按空格递进
4. 抄完显示「抄写完成」

## 设置项

| 设置 | 说明 |
|------|------|
| 简约模式 | 关闭背景地图与光晕效果，减少性能开销 |
| 单行最大字数 | 每句最多显示字数，超出强制切分（5–50，默认 10） |
| 点击以递进 | 开启后点击页面空白处即可前进 |

## 技术

纯静态网页，零依赖，零构建步骤。

该网页完全在用户端侧运行，所有用户的信息不经过服务器。
